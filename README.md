# Café Libretto – Website

Website for Café Libretto, Frankfurt am Main.

**Developed by [ByteTechnik.de](https://bytetechnik.de)**

## How to run the project locally

Requirements: Node.js and npm – [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Clone the repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
